# Инструкция по добавлению шрифта Minecraft

Для корректной работы сайта необходимо добавить шрифт Minecraft в эту папку.

## Как добавить шрифт:

1. Скачайте шрифт Minecraft.ttf из любого из этих источников:
   - https://www.dafont.com/minecraft.font
   - https://fonts.gg/minecraft
   - https://fontmeme.com/minecraft-font/

2. Поместите файл `Minecraft.ttf` в эту папку (website/fonts/)

3. Убедитесь, что название файла точно соответствует `minecraft.ttf` (с маленькой буквы)

## Важно:
- Шрифт Minecraft используется для стилизации сайта в стиле игры
- В CSS файле шрифт уже подключен через @font-face
- Если не добавить шрифт, сайт будет использовать резервные шрифты (Segoe UI, Tahoma, и т.д.)

Сайт будет работать без шрифта Minecraft, но для создания аутентичного стиля Minecraft рекомендуется его добавить.
